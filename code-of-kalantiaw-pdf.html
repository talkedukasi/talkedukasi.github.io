1. The Code of Kalantiaw​


Answer:

The Code of Rajah Kalantiaw was a supposed legal code in the epic history Maragtas that is said to have been written in 1433 by Datu Kalantiaw, a chief on the island of Negros in the Philippines. The code is now believed by many historians to have been a hoax and that it had actually been written in 1913 by Jose E. Marco as a part of his historical fiction Las antiguas leyendas de la Isla de Negros (English: The Ancient Legends of the Island of Negros), which he attributed to a priest named Jose Maria Pavon.

In 1990, Philippine historian Teodoro Agoncillo described the code as "a disputed document."[1] Despite doubts on its authenticity, some history texts continue to present it as historical fact.[2]

11. What made william henry scott conclude that the kalantiaw code is a fake historical code?


Answer:

All information about Kalantiaw was written by and traced back to Jose Marco.

Explanation:

The Kalantiaw Code is a fictional or mythical set of rules that is believed to be written by Datu Kalantiaw.

When William Henry Scott was working on his thesis in 1965, Mauro Garcia advised him to cover the events of the pre-colonial Philippines. During his research, he encountered the Kalantiaw code and started investigating the legitimacy of Jose Marco's creation.

In his investigation, he searched through archives, asked for the opinion of fellow historians and Marco's friends, and looked through the references of his college books. He discovered that all the information about the Kalantiaw is written by Jose Marco, not by some person in the early 1400s.

With this information, Scott included that the Kalantiaw code is fake.

The Code of Kalantiyaw is a supposed pre-Spanish Philippine criminal code that was found on the island of Panay in 1614 and is said to have been written in 1433. José E. Marco, the code's "discoverer," was later exposed as a dealer of historical forgeries, according to later study.

17. the documents where the kalantiaw code was included​


Answer:

Datu Kalantiaw (Rajah Bendahara Kalantiaw) (sometimes spelled Kalantiao) was once considered an important part of Philippine history as the one who created the first legal code in the Philippines, known as the Code of Kalantiaw in 1433. He was considered by Filipinos, particularly the Visayans, as the third head of Panay (an island in Visayas with four provinces: Aklan, Antique, Capiz and Iloilo). He was a source of fierce Filipino and Visayan pride for decades, until his authenticity was debunked by Historian William Henry Scott in his PhD thesis, Critical Study of the Prehispanic Source Materials for the Study of Philippine History relegating the once legendary historical figure into a mythical Filipino character or an urban legend.

There are several similarities between the Kalantiaw Code and the 1987 Philippine Constitution. Both documents discuss the concept of equality, with the Kalantiaw Code stating that "All persons shall be equal before the law" and the Constitution affirming that "The State recognizes the dignity of every human person and guarantees full respect for human rights."
