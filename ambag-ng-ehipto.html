12. Ano ang mahalangang ambag ng kabihasnang ehipto sa larangan ng agrikultura?


Answer:

Ambag ng mga Sibilisasyong Ehipto sa Agrikultura

Ang sibilisasyong Ehipto ang isa sa mga pinakatanyag at pinakamalakas na sibilisasyon noong unang panahon. Sila din ang unang nagsulong ng malakihang industriya ng agrikultura, at utang nila sa Ilog Nile ang pagiging mataba ng kanilang mga lupain. Dahil dito, nalinang ng husto ng mga taga-Ehipto ang agrikultura, at narito ang ilan sa mga naging kontribusyon ng kanilang sibilisasyon:

-Ang pagtatayo ng isang malaking sistema ng mga irigasyon.

-Pagbuo ng malalaking imbakan ng tubig na gagamitin bilang supply kung magkulang ito.

-Pagtatanim ng iba’t-ibang uri ng pananim, kabilang na ang trigo, barley, flax, at papyrus.

-Ang pagdedevelop ng horticulture, at pagkuha ng mga prutas at gulay mula sa mga harding nabuo dahil dito.

16. Ano ang mahalagang ambag ng Kabihasnang Ehipto sa larangan ng Agrikultura?​


Ambag ng Ehipto sa agrikultura

Answer:

Ang kabihasnang Ehipto ay malaki ang naging ambag sa larangan ng Agrikultura. Ayon sa mga historians, isa sila sa mga unang grupo ng tao kung saan nagsimula ang konspeto ng pagtatanim. Dahil ang ilog Nile ay matatagpuan dito, ginamit nila ito upang gawing patubig at nagdebelop din sila ng sistema ng irigasyon upang mapadali ang kanilang pagtatanim.

Bukod dito, sa kanila rin nagsimula ang horticulture o ang pag-aalaga ng mga bulaklak at paggawa ng gardens. Iba-ibang produkto ang kanilang itinanim kabilang ang mga prutas, grains at gulay. Nagtanim din sila ng iba't ibang halamang herbal.

Answer:

Ilan sa mga naibigay na tulong ng nasabing kabihasnan ay ang pagkakagawa ng sistemang irigasyon, pagkakagawa ng imbakan ng tubig, at ang pagkatuto sa pagtatanim ng iba’t ibang klase ng halaman.

Ang pagkakagawa ng Sistema ng irigasyon. Ang irigasyon ay ang daluyan ng tubig na kung saan ginagamit ito upang mabigyan ng sapat na tubig pandilig sa mga pananim.


22. ano angmahalagang ambag ng kabihasnang ehipto sa larangan ng agrikultura​


Answer:

﻿Ang kabihasnang ehipto ay nakapag-ambag ng malaki sa usaping agrikultura sapagkat napaunlad nito ang pamamaraan ng pagtatanim.

Ilan sa mga naibigay na tulong ng nasabing kabihasnan ay ang pagkakagawa ng sistemang irigasyon, pagkakagawa ng imbakan ng tubig, at ang pagkatuto sa pagtatanim ng iba’t ibang klase ng halaman.

Ang pagkakagawa ng Sistema ng irigasyon. Ang irigasyon ay ang daluyan ng tubig na kung saan ginagamit ito upang mabigyan ng sapat na tubig pandilig sa mga pananim.

Sa kabilang banda, ang pagkakagawa ng imbakan ng tubig ay nakatutulong sa pagtatanim upang hindi kapusin ng kakailangang tubig para sa gawaing ito.

Samantala, mas napaunlad ang agrikultura sa tulong ng kabihasnang ito dahil ang mga mamamayan ay natutong magtanim pa ng ibang klase ng halaman.
