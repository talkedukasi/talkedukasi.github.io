Create two (2) word problems related to permutation with solution and answer.

[tex]\:[/tex]

Question: A class is to select a president, a vice president, a secretary, and a treasurer from 7 class members. How many arrangements of class officers are possible?

[tex]\:[/tex]

Solution:

The number of permutations of 7 class members taken 4 at a time is given by [tex]\sf{_7P_4}[/tex]. Thus,

[tex]\sf{_7P_4}[/tex] = [tex]\large\sf{ \frac{7!}{(7\:-\:4)!}}[/tex]

= [tex]\large\sf{ \frac{7!}{3!}}[/tex]
