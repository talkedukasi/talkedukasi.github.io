19. Nakapagbibigay ng mga halimbawa ng conative, phatic, emotive at referential na gamit ng wika a komunikayon. Panuto: Sumulat ng tig-dalawang pangunguap a conative, phatic, emotive at referential na gamit ng wika.


Answer:

Nakapagbibigay ng mga halimbawa ng conative, phatic, emotive at referential na gamit ng wika a komunikayon. Panuto: Sumulat ng tig-dalawang pangunguap a conative, phatic, emotive at referential na gamit ng wika.

Ang mga halimbawa ng mga pangungusap na may conative, phatic, emotive at referential na gamit ng wika ay maaaring magkahawig na sumusunod:

Conative: "Tumigil ka na sa pag-uusap!" (Ito ay isang pangungusap na may layunin na payuhan o utusan ang isang tao na tumigil sa pakikipag-usap.)

Phatic: "Kumusta ka na?" (Ito ay isang pangungusap na ginagamit lamang upang magpakita ng interes sa kalagayan ng isang tao.)

Emotive: "Nakakalungkot isipin na hindi tayo magkakasama muli." (Ito ay isang pangungusap na nagpapahayag ng damdamin o emosyon ng taong nagsasalita.)

Referential: "Ang Maynila ay ang pangunahing lungsod ng Pilipinas." (Ito ay isang pangungusap na nagpapaliwanag o nagpapakahulugan ng isang bagay o konsepto.)


20. 7. Which element of Jacobson's model of communication is associated with the referential function? A. Context B. Code C. Sender D. Receiver​


Answer:

The element of Jacobson's model of communication associated with the referential function is B. Code.

Explanation:

The referential function of communication refers to the transmission of information that serves the purpose of referring to or representing an object, event, or idea. In Jacobson's model of communication, the code refers to the signs, symbols, or language used to transmit information from the sender to the receiver. The code is the means by which the referential function of communication is achieved.

In other words, the code is what enables the transmission of information that refers to or represents something. The code can be any form of communication, such as verbal or written language, mathematical symbols, images, or gestures. The use of an appropriate code helps to ensure that the message being transmitted is understood by the receiver in the way it was intended by the sender.

In Jacobson's model, the code is one of the key elements that contribute to the success or failure of communication. The choice of code, its clarity and accuracy, and the ability of the receiver to interpret it correctly are all factors that determine whether the referential function of communication is achieved effectively.
