3. ano ang TEKSTONG DESKRIPTIBO ?


Answer:

Sinasabing ang teksto ay deskriptibo kung ito ay uri ng tekstong naglalarawan. Naglalaman ito ng impormasyong ginagamitan ng mga salitang pantukoy sa katangian ng isang tao, bagay, hayop, lugar, o pangyayari. Mayaman sa mga salitang pang-uri o pang-abay ang mga tekstong deskriptibo. Nakatutulong kasi ito sa malinaw na pagtukoy sa mga katangian.

5. ano ang katangian ng tekstong deskriptibo​


KATANGIAN NG TEKSTONG DESKRIPTIBO

Ano ba ang katangian ng isang tekstong deskriptibo?

Mahalaga na sa pagbuo ng isang teksto ay may relasyon nito sa iba pang uri ng teksto. Lagi nitong kabahagi ang tekstong naratibo sa paglalarawan. Inilalarawan nito ang mga tauhan, ang tagpuan, ang damdamin, at ang tono ng pagsasalaysay. Ito ay ginagamit din sa paglalarawan sa panig na pinaniniwalaan at ipinaglalaban sa tekstong persuwisibo o sa paglahad kung paano magagawa o mabubuo nang maayos ang isang bagay para sa tekstong prosidyural. Ang tekstong ito bihirang magamit nang walang kabahagi ng iba pang uri ng teksto. Ang tekstong deskriptibo ay gumagamit ng cohesive device upang mas maging malinaw ang uri ng anumang uri ng tekstong susulatin.

Mga cohesive devices sa tekstong deskriptibo:

1. Reperensiya

a. Anapora

b. Katapora

2. Substitusyon

3. Ellipsis

4. Pang-ugnay

5. Kohesyong leksikal

14. Ano ang Tekstong Deskriptibo? Ipaliwanag​


Answer:

Ang tekstong deskriptibo ay nagtataglay ng mga kaukulang detalye sa katangian ng isang tao, lugar, bagay, o pangyayari. Ang tekstong ito ay laging sumasagot sa tanong na “Ano“. Sa Tekstong Deskriptibo, kahit hindi ka pintor ay makakabuo ka ng isang larawan gamit ang mga salitang magmamarka sa damdamin at isipan ng mambabasa. Upang mailarawan at mabigyang buhay sa imahinasyon ng mambabasa ang isang tauhan, tagpuan, bagay, galaw o kilos,karaniwang gumagamit ang may akda ng pang-uri at pang abay. Gumagamit din ng manunulat ng iba pang paglalarawan tulad ng panggalan, pandiwa, tayutay, pagwawangis, pagsasatao at iba pa. Ang tekstong deskriptibo ay laging kabahagi ng iba pang uri ng teksto, partikular sa tekstong naratibo.

23. ano ang halimbawa ng tekstong deskriptibo


Ang tesktong deskriptibo ay uri ng akda na madalas isinusulat para sa mga malikhaing katha. Ito ay maaaring magtampok ng paglalarawan ng mga tao, panahon, at mga lugar upang hikayatin ang mambabasa na maglakbay sa malikhaing mundo ng mga iba’t ibang kuwento o kaya ay magbigay ng konteksto sa paglalarawan ng isang tao. Ang mga alamat, kwentong bayan, character sketch at iba pang uri ng mga kathang-isip na sulatin ay maaring maging halimbawa ng tesktong deskritibo.

28. ano ang layunin ng tekstong deskriptibo?


Answer:

Ang layunin ng tekstong deskriptibo ay ang makapagpamalas sa isip ng tagapakinig o mambabasa ng isang malinaw at buong larawan.


29. ano ang kahalagahan ng tekstong deskriptibo?​


Answer:

Mahalaga ang paglalarawan sa teksto upang mas maintindihan ng mambabasa ang imahe o mensahe na nais ipahayag ng manunulat. Nakakatulong ito upang mapalawak ang imahinasyon ng mambabasa. Mas madaling maiintindihan ang tekstong binabasa kung malinaw ang paglalarawan sa mga ito.


30. ano ang katangian ng tekstong deskriptibo​


Answer:

Ang descriptive text ay isang tekstong nagsasabi kung ano ang hitsura ng isang tao o isang bagay. Ang layunin nito ay ilarawan at ihayag ang isang partikular na tao, lugar, o bagay. Ang bahaging ito sa unang talata - ang mithiin ay matukoy ang isang bagay na ilalarawan o ilarawan.
