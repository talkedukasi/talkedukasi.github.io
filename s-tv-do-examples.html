Subject-Transitive verb-Indirect object-Direct object

Transitive verb - a verb that requires both a subject and a direct object

Indirect Object - word that tells whom the action is done (*answers the questions "for whom" or "to whom"

Direct object - direct receiver of the action done by the subject

Example Sentence: 1. Mary sent a copy of the report.

                                 2. She bakes me a birthday cake.

Subject (S) - Mary / She

Transitive verb (TV) - sent / bakes

Indirect object (IO) - Peter / me

Direct object (DO) - report / birthday cake.

30. give 5 examples of ''subject-transitive verb indirect object direct object(s-tv-io-do) pattern' ''


Answer:

Subject-Transitive Verb-Indirect Object-Direct Object Sentence Pattern

Subject-Transitive Verb-Indirect Object-Direct Object is one of the sentence patterns. Let us define each element of a sentence.

Subject - the one who does the action or the topic of the sentence.

Transitive Verb - verb that requires both a subject and a direct object.

Indirect Object - precedes the direct object and tells to whom or for whom the action of the verb is done and who is receiving the direct object.

Direct Object - receives the action done by the subject.

Examples of Subject-Transitive Verb-Indirect Object-Direct Object

Toby bought sister some flowers.

We gave teacher a birthday cake.

I teach the students Math.

Roger sent Ruby a letter.

I gave mother a new pair of slippers.
