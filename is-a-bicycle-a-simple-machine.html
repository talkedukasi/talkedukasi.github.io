1. Is a bicycle a simple machine


Yes because it hasLevers: Shifters, crankset (the pedal levers), derailleurs, handlebars, freewheel assembly, brakes. Wheel and axel: The wheels, pedals, crankset, cassette Pulleys: Parts of the shifting mechanisms, braking mechanisms, and the drive train (chain on gears). Screws: (Hold parts together). Wedges: The teeth on the gears. Not common, but some gooseneck assemblies (where the handlebars attach to the front fork tube) may employ a wedge to tighten the connection. If you have a repair kit on your bike, tire levers are also both levers and wedges.

4. Demonstrate how simple machines help make work easier and faster: Lever Inclined plane Wedge Screw. . Wheel and axle, eg, bicycle Pulley?​


Answer:

Lever: A lever is a simple machine that is used to make work easier and faster. A lever is a bar that is hinged at one end and is used to lift or move heavy objects. The lever is able to do this because it is able to magnify the force that is applied to it. Inclined Plane: An inclined plane is a simple machine that is used to make work easier and faster. An inclined plane is a ramp that is used to raise or lower a heavy object. The inclined plane is able to do this because it increases the distance that the object is moved. Wedge: A wedge is a simple machine that is used to make work easier and faster. A wedge is a two-sided object that is used to split or pry objects apart. The wedge is able to do this because it increases the force that is applied to it. Screw: A screw is a simple machine that is used to make work easier and faster. A screw is a rod with a spiral groove that is used to join or hold objects together. The screw is able to do this because it increases the friction between the objects.
