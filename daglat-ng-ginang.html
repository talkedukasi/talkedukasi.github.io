Sa buhay, minsan nakakatagpo ka ng napakahabang salita. Ang mga pagdadaglat ay mga pinaikling anyo at tashid ng isa o higit pang mga titik. Ang mga pagdadaglat ay may iba't ibang uri at paraan ng pagsulat, depende sa kanilang paggamit.

Ang pagdadaglat ay isang anyo na nakasalalay sa mga pantig na binubuo ng isa o higit pang mga pantig, na maaaring bigyang-kahulugan bilang isang buod. Ang mga pagdadaglat ay karaniwang ginagamit upang paikliin ang mga pangalan ng mga tao, degree, posisyon, trabaho, institusyon, yunit ng sukat, at mga pangungusap.

Mga Uri ng Daglat

Ang iba't ibang mga pagdadaglat ay ang mga sumusunod:

Pagpapaikli ng pangalan ng tao, pangalan ng tao, tao, posisyon o grupo

Ang pagdadaglat na ito ay sinusundan ng isang sign letter.

Daglat para sa opisyal na pangalan ng ahensya ng pabahay, ahensya, organisasyon, at ang pangalan ng opisyal na dokumento.

Ang ganitong uri ng pantig ay binubuo ng mga titik sa simula ng mga salita. Ang pagsulat ay gumagamit ng malalaking titik na walang mettului na may mga marka ng titik.

Pagpapaikli ng katana sa isang pulong o panayam. Ang ganitong uri ng pagdadaglat ay isang kumbinasyon ng mga titik na sinusundan ng isang marka ng titik.Mga karaniwang pagdadaglat

Ang mga karaniwang pagdadaglat ay binubuo ng dalawang titik at sinusundan ng isang marka ng titik sa bawat titik. Ang pagdadaglat na ito ay ginagamit para sa mga layunin ng pagsusulatan.

Mga pagdadaglat para sa mga kaalyado na drain, kaliskis, kemikal, sukat, at salita.

Ang pagdadaglat na ito ay ginagamit nang hindi sinusundan ng isang liham.
