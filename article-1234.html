20. which of the following are not polygons?​ 1234


1 and 2?

A polygon is a closed shape with straight sides. To identify a polygon, you need to check if it meets these criteria:

1. It has three or more sides.

2. All sides are straight.

3. All angles are less than 180 degrees.

4. It is a closed shape.

If a shape does not meet these criteria, it is not a polygon. For example, a circle is not a polygon because it has curved sides. Similarly, a line segment is not a polygon because it has only two sides.
