20. Differentiate between permutation in general (n objects taken r at a time ,circular permutation ,and distinguish permutation (when some object are anlike


A permutation, in general, is an arrangement of objects where order matters. If two sets containing the same elements but are ordered differently, then they are two different permutations.

A circular permutation is a kind of permutation where instead of elements are being arranged in a line, they are arranged in a circle. Because of this, permutations are fewer. A set of n elements have n! permutations but in a circular permutation, only (n-1)! permutations are formed. We can look at this number in this way: n!/n.
n! is the number of permutations in a line.
Since we want the number of circular permutations, we have to divide this number by n to eliminate duplicates. Notice that in a circular permutation, if every element were to shift towards the same direction, the permutation would still be the same as before.

In a set where an element appears more than once, the number of permutations is given by:
n!/(p!q!) where p,q are the number of times an elements is repeated. This can be thought of as eliminating repetition by finding the number of permutations p,q can have if relation to their position.
