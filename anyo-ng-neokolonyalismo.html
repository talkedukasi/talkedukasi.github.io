8. ANO ano ang anyo ng neokolonyalismo? ​


Anyo ng Neokolonyalismo

Ekonomiko - Ito ang anyo ng neokolonyalismo na napalaganap sa pamamagitang ng tulong tungo sa pagpapaunlad sa kalagayang pangkabuhayan ng bansa.

Kultural - Ito naman ang anyo ng neokolonyalismo na nabago ang pananaw sa mga bagay na likas na angkin ng bansa buhat ng mga ipinakilala ng dayuhan.

Kahulugan ng Neokolonyalismo

Ang neokolonyalismo ay binubuo ng dalawang salita, neo at kolonyalismo. Ang ibig sabihin ng neo ay bago. Ang ibig sabihin naman ng kolonyalismo ay pananakop ng bansa upang makakuha ng pangangailangan. Mula sa dalawang salitang ito, ang neokolonyalismo ay tumutukoy sa makabagong pangkultura at panlipunang impluwensya ng mga mananakop. Ito ang makabagong imperyalismo.

Dahilan ng Neokolonyalismo

Pagtanggap ng tulong o donasyonPaggamit ang impluwensyaSuportang militarPakikipag-ugnayan sa programa ng ibang bansa

24. Mga anyo at pamamaraan ng neokolonyalismo


Ang salitang neokolonyalismo ay ginagamit upang ilarawan ang isang makabagong uri ng kolonyalismo. Dito, kinokontrol ng isang kolonisador na bansa ang pamumuhay ng mga tao sa bansang kolonya. Pinanghihimasukan din ng mga kolonisador ang pamamahala at gobyerno ng kanilang kolonya.

Mga anyo o uri ng neokolonyalismo

1. Ekonomiko - panghihimasok sa

pamamalakad ng ekonomiya ng isang bansa

2. Kultural - pagyakap ng isang bansa

sa mga banyagang kultura

3. Gobyerno - pinanghihimasukan ang

pamamahala sa isang bansa

Ang mga pamamaraang ginamit ng neokolonyalismo upang makuha ang kanilang gusto sa malayang bansa ay kinabibilangan ng mga uring pang-ekonomiya at pangkultura. May mga pagkakataong ginamit din ang militar at ang mga pailalim na gawain ng mga instutusyong pang-espiya.
