SOLVING FUNCTIONS FROM WORD PROBLEMS

Conclusion answers to the given questions are as follows:

a. The function M(n) is M(n) =  P60, 000/n

b. If the organization has eight committees, each committee gets P7,500

Further explanation

Solving Function from Word Problems:

To solve the given problem, we must know how to construct word problems into the form of a linear function or any type of function.

Here are the steps:

Read the problem carefully to find the key details. Define descriptive variables to represent the quantities. Look for information that provides values for the variables or specific functional model components.Identify a solution method from the provided information to what we are trying to find.Write a formula for the function.Use the formula to solve or evaluate the function.

Example:

Atika attends math lessons with a mandatory monthly fee of P10,000 and a cost per meeting of P5,000. If Atika attends 4 meetings for a month, then the tutoring fee that Atika must pay is ...

Solution:

Let:

G(n) = a function that would give the amount of tutoring fee that Atika must pay

n = the number of meetings

Model construction for the function:

The amount of tutoring = mandatory monthly fee + (cost per meeting x number of meetings)

Then, the formula for the function

G(n) =  P10,000 + (P5,000 x n)

The tutoring fee that Atika must pay If Atika attends 4 meetings for a month is G(4), so evaluate G(4)

G(4) =  P10,000 + (P5,000 x 4)G(4) =  P10,000 + P20,000G(4) = P30,000

Given Problem:

The budget of a university organization = P60, 000.

The budget is split evenly among the committees.

a.  Construct a function M(n) that would give the amount of money each of the n numbers of committees would receive.

Let:

M(n) = a function that would give the amount of money each of the n numbers of committees would receive

n = the number of committees would get

Model construction for the function:

Amount of money each committee would receive = the budget of a university/the number of committees

If they have a budget of P60, 000, the function becomes:

M(n) =  P60, 000/n

b. If the organization has eight committees, how much would each committee have?

The organization has eight committees,

n = 8

What is asked is to find M(8).

Because the n value is a number 8, substitute to M(n)

M(8) =  P60, 000/8

Simplify M(8) to get:

M(8) =  P7,500

Therefore, the solution is that each committee gets P7,500.
