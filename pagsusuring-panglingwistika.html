15. Pagsusuring pangnilalaman


Answer:

Ang pagsusuri ng nilalaman ay isang tool sa pananaliksik na ginagamit upang matukoy ang pagkakaroon ng ilang mga salita, tema, o konsepto sa loob ng ilang ibinigay na data ng husay (ibig sabihin, teksto). Gamit ang pagsusuri ng nilalaman, masusuri at masusuri ng mga mananaliksik ang presensya, mga kahulugan at mga kaugnayan ng mga partikular na salita, tema, o konsepto.

Mayroong dalawang pangkalahatang uri ng pagsusuri sa nilalaman: pagsusuri ng konsepto at pagsusuri sa relasyon. Tinutukoy ng pagsusuri sa konsepto ang pagkakaroon at dalas ng mga konsepto sa isang teksto. Ang relational analysis ay nagpapaunlad ng konseptwal na pagsusuri sa pamamagitan ng pagsusuri sa mga relasyon sa pagitan ng mga konsepto sa isang teksto. Ang bawat uri ng pagsusuri ay maaaring humantong sa iba't ibang resulta, konklusyon, interpretasyon at kahulugan.

Ang mga kalakasan ng pagsusuri ng nilalaman ay ito ay isang maaasahang paraan upang pag-aralan ang data ng husay dahil ang mga yunit ng coding ay hindi bukas sa interpretasyon at sa gayon ay inilalapat sa parehong paraan sa paglipas ng panahon at sa iba't ibang mga mananaliksik. Ito ay isang madaling pamamaraan na gamitin at hindi masyadong nakakaubos ng oras.

28. Ano ang pagsusuring pampanitikan?​


Answer:

Ang pagsusuring pampanitikan ay isang pag-aaral, pagtalakay, pagsusuri at pagpapaliwanag ng panitikan. Ito ay may dalawang sangay. Ang unang sangay ay ang Pagdulog. Ang mga uri nito ay pormalistiko o pang-anyo. Ang pangalawa ay ang moralistiko. Ang pangatlo ay ang sikolohikal. at ang huling uri ay sosyolohikal-panlipunan.

Ang pangalawang uri ay ang Pananalig. Binubuo ito ng maraming uri. Ang mga uri nito ay klasisismo, romantisismo, realismo, naturalismo, impresyunalismo, ekspresyunalismo, simbolismo, eksistensiyalismo, at peminismo.

Answer:
