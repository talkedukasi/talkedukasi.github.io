A static website, also called a flat or stationary page, is displayed in a web browser exactly as it is stored. It contains web pages with fixed content coded in HTML and stored on a web server. It does not change, it stays the same, or "static" for every viewer of the site.

On the other hand, a dynamic website contains web pages that are generated in real-time. These pages include web scripting code, such as PHP or ASP. When a dynamic page is accessed, the code within the page is parsed on the Web server and the resulting HTML is sent to the client's Web browser. This allows more personalized browsing for all users.

A static character refers to the character that does not undergo change in terms of his or her personality and perspective while a dynamic character refers to the character that undergo transition and change because of his or her experience and challenges encountered.

It is so because her character has a transition from being reserved and scared of her own power to expressing her power by creating a castle and a beautiful bridge made of ice.
