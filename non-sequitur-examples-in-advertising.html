Syllogism is an argument form having a major and minor premise and ending with a conclusion. Non-sequitur is a fallacy of syllogism where there is a predictable sequence that help us create a valid argument.

Examples are:

affirming the consequent and denying the antecedent.

Consequent Affirmation – If A then B. B is true, so A is true.

Antecedent Denial – If A then B. A is false, so B is false.

Halimbawa:

"Ang mga rosas ay kulay pula. Kaya naman ang mga halaman ay mahalaga sa kalikasan."
