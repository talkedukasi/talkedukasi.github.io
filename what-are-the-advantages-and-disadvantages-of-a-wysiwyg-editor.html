9. what are the advantages and disadvantages of WYSIWYG editor​


WYSIWYG (pronounced wiz-ee-wig) is an acronym for "what you see is what you get." It has some advantages as well as disadvantages as follows:

WYSIWYG editors enable users to manipulate the content or layout without having to type any commands. For example, when users write a document using a word processor, it uses WYSIWYG, as what they create, format and edit is replicated in the printed document or PDF file. Thus the advantages of WYSIWYG editor are:

Simple, quick, no specific abilities necessary.

It's a lot of fun to make web sites this way. You may experiment with colors and layouts on your own. Unless you are attempting to clone another web page, the web page is likely to be unique.

You may let your imagination go wild. It is now easier and faster to construct web sites. You may modify the layout of a page by dragging the elements to their new placements.

Instead of searching for and inserting within HTML elements or utilizing a sophisticated content management system, you may make changes to content (text or pictures) via a WYSIWYG editor.

Meanwhile the disadvantages are it takes as long to learn how to use them as it does to learn to program, and the worst part is that you will become lost in inconceivable dead ends.

You will be labeled a "perpetual amateur" since you will have learnt NOTHING about programming (and never be taken seriously).

A lot of HTML code generated by editors is redundant, resulting in big file sizes for web pages, and as you know, a large file downloads slower than a smaller one. In addition, search engines must wade through all of this code to get to the meat of the website - the actual text contents.
