11. Kai pays a convenience fee when he reserves tickets online. The table below shows him the total cost of his purchase for different number of tickets. Tickets, x Cost, y 2 $32.00 3 $44.50 6 $82.00 Which function can be used to find y, the cost of the tickets for x number of tickets?


Answer:

From the table, we can see that the cost of the tickets increases as the number of tickets increases. This indicates that the cost of the tickets is a linear function of the number of tickets.

A linear function is a function of the form y = mx + b, where m is the slope of the line and b is the y-intercept. The slope represents the rate at which the output value (y) changes with respect to the input value (x). The y-intercept is the point where the line crosses the y-axis.

To find the linear function that represents the cost of the tickets, we can use the slope-intercept form of a linear equation, which is y = mx + b. We can find the slope (m) and y-intercept (b) by using two points from the table.

For example, we can use the points (2, 32) and (3, 44.50) to find the slope and y-intercept. The slope is (44.50 - 32) / (3 - 2) = 12.50 / 1 = 12.50. The y-intercept is 32 - (12.50)(2) = 32 - 25 = 7.

Substituting these values into the slope-intercept form, we get: y = 12.50x + 7

This function can be used to find the cost of the tickets for any number of tickets. For example, the cost of 4 tickets is y = 12.50(4) + 7 = 50 + 7 = $57.00. The cost of 5 tickets is y = 12.50(5) + 7 = 62.50 + 7 = $69.50.

A student ticket cost $1. An adult ticket costs $2.50. Total ticket sales are $164. How many student tickets were sold?


Answer:

There are 59 student tickets sold.

Step-by-step explanation:

Number of tickets:

Adult tickets: xStudent tickets: yTotal tickets sold:  101

Equation (1):

x + y = 101

y = 101-x  (student tickets)

Cost per ticket:

Cost per adult ticket: $2.50xCost per student ticket: $1yTotal ticket sales: $164

Equation (2):

2.50x + 1y = 164

or

2.5x + y = 164

Solve by substitution where y=101-x:

2.5x + y = 164

2.5x + (101-x) = 164

2.5x -x + 101 = 164

Find x, the number of adult tickets:

1.5x = 164 - 101

1.5x/1.5 = 63/1.5

x = 42  ⇒  number of adult tickets

Find y, the number of student tickets:

y = 101 - 42

y = 59  ⇒  number of student tickets

There are 59 student tickets sold.

Check:

x = 42  and y = 59

Equation (1):

x + y = 101

42 + 59 = 101

101 = 101  (True)

Equation (2):

2.5x + 1y = 164

2.5(42) + (59) = 164

105 + 59 = 164

164 = 164 (True)
