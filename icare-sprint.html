24. A 100-meter sprint is a race using only the straight side of a racetrack. A 400-meter sprint is a race that makes one complete lap around a racetrack. Why are velocity and speed the same for a 100-m sprint but not a 400-m sprint?


Answer:

Speed uses distance and velocity uses displacement in its calculation.

For 100 m race, distance = displacement. Hence speed = velocity

For 400m race, distance ≠ displacement. distance = 400m whereas displacement = 0m. Hence speed ≠ velocity

30. a 100-meter sprint is a race using only the straight side of a racetrack. a 400-meter sprint is a race that makes one complete lap around a race. why are velocity and speed the same for a 100-m sprint but not a 400-m sprint


Velocity and speed the same for a 100-m sprint but not a 400-m sprint because for a 100-m sprint runners move in the same direction

Further explanation

Vector is a quantity that has a value and direction

Vector can be symbolized in the form of directed line segments

[tex]\large {\boxed {\bold{\overrightarrow {A}}}[/tex]

while the length of the vector is denoted by | a |

 Vectors can be written in the form of sequential pairs which shows their coordinates in the Cartesian plane: a (a₁, a₂)

 with length

[tex]\large {\boxed {\bold {| a | = \sqrt {a_1 ^ 2 + a_2 ^ 2}}}[/tex]

If the direction of the vector is reversed, we get the vector -a which has the same length but in the opposite direction

A vector has a direction and a magnitude, while a scalar has only magnitude.

Examples of scalars are: length, mass, time, speed

Examples of vectors are: force, acceleration, velocity

So velocity has direction while speed has no direction.

Speed is the distance divided by the time taken. While the velocity is the rate of displacement per unit of time.

The displacement is a vector quantity while the distance is a scalar quantity, the distance is the length of the path traveled by an object in a certain time interval while the displacement is a change in the position of the object in a certain time interval

When a runner does a 100 m sprint then the track used is only a straight line in the same direction so that the speed and speed have the same value

When runners do 400 m sprints, make runners will pass a rotating track so that it has a different direction on each track so that the speed and velocity is different. The velocity on this track will be calculated based on the runner's displacement while the speed is calculated from the total distance from the runner within a specified time interval
