25. salitang ugat ng susuguin


Ang salitang ugat ng susuguin ay sugo.

Ang ibig sabihin ng salitang ugat ay mga salita na hindi na dinadagdagan pa, ang mga salitang ito ay buo na ang kilos.
Halimbawa:
Umunlad- ang salitang ugat ay unlad. Hulaan- ang salitang ugat ay hula. Simsimin- ang salitang ugat ay simsim. Sunugin- ang salitang ugat ay sunog. Ipatong- ang salitang ugat ay patong.
