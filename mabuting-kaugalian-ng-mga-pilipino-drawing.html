Ang mga ito ay kabilang sa 4 Core Values, inilalarawan nito ang mga mabubuting kaugalian ng bawat Pilipino. Kasama an Core Values sa DepEd Vision.

Ang DepEd Vision ay isinasagawa sa mga Flag Ceremony ng mga Paaralan mula K-12. Ito ay ang panunumpa ng pagiging Maka-bansa, ang mga tao na nanunumpa nito ay kailangang sundin ng kanilang isinumpa sa Watawat ng Pilipinas.

30. alin sa mga kaugaliang ito ang naiiba sa kaugalian ng mga pilipino?sang-ayon kaba rito?bakit?
